# Public Advisory Overlap Attribution

This fixture contains six public advisory records selected from pinned GitHub Advisory Database and PyPA Advisory Database commits. GitHub Advisory Database records are distributed under CC-BY-4.0. PyPA Advisory Database records are distributed under CC-BY-4.0. The fixture keeps upstream record identifiers and paths so that evaluators can trace every normalized claim to its public source. PyPI release-universe rows are derived from the public PyPI JSON API response cached for the package at fixture construction time.
